# Bankloan

U tabeli imamo 700 korisnika za koje znamo da li su dobili kredit ili ne, treba odrediti da li novih 150 korisnika treba da dobije kredit.


# Init project

```
pyenv local 3.10 

python3 -m venv env

source env/bin/activate

pip install -r requirements.txt

```


# Run script

```
python main.py
```
